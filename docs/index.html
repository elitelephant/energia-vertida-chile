El norte de Chile derrocha energía

Chile tiene un desafío. El norte genera más energía de la que puede consumir. El sol es abundante y las plantas generan a plena capacidad. Pero la red eléctrica que conecta el norte con el sur, donde está la mayor parte del consumo, no tiene capacidad suficiente para transportar todo ese exceso. Cuando eso ocurre, la energía simplemente se pierde. A eso se le llama vertimiento.
